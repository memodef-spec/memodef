# decisions/

Strategist decision artifacts capturing the rationale behind template-library inclusion calls and accepted spec proposals.

## File patterns

### `<id>.md` — template inclusion decisions

One decision artifact per accepted template promotion.

### `<id>-v<N>.md` — version-bump retrofit decisions

When an existing template is bumped to a new major version.

### `proposal-<name>.md` — spec proposal decisions

When a spec proposal in `../proposals/` is decided.

### `bootstrap-<event>.md` — bootstrap-event records

For governance events outside the inclusion / proposal categories (e.g., the initial bootstrap deviation record).
